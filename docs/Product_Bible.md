# Notebook Structure

Every notebook contains:

- Month
- Week
- Day

No templates.

No notebook types.

---

# MVP Features

- Notebook Library
- Month View
- Week View
- Day View
- Apple Pencil Drawing
- Text Tool
- Local Storage

---

# Future Features

- Export
- iCloud Sync
- Premium Paper Styles
- Premium Brushes
- Stickers

Community photo submission will be considered after the core experience is complete.

---

# Design Philosophy

The notebook is the product.

The interface should disappear while writing.

Everything should feel calm, tactile and intentional.

---

# Success Metric

A user should enjoy opening Leaf even before writing anything.
